# Нормализация базы данных — Личный финансовый менеджер

## Нормализация до 1NF
**Решено:** Все атрибуты атомарны.  
**Изменения:** Категории вынесены в отдельную таблицу `Categories`.

## Нормализация до 2NF
**Решено:** Устранены частичные зависимости.  
**Изменения:** Данные пользователей и счетов разделены на отдельные таблицы `Users` и `Accounts`.

## Нормализация до 3NF
**Решено:** Устранены транзитивные зависимости.  
**Результат:** Каждая таблица содержит только атрибуты, зависящие от её первичного ключа.  
**Таблицы:**  
- `Users` (пользователи)  
- `Accounts` (счета)  
- `Categories` (категории)  
- `Transactions` (транзакции)

## Вывод
База данных спроектирована и нормализована до третьей нормальной формы (3NF).  
ER-диаграммы сохранены в папке `/docs/database/` в форматах `.svg` и `.png`.
